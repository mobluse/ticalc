TI-82, TI-82 STATS, TI-83, TI-83 Plus & TI-84 Plus Programs & Resources

[TEXAS INSTRUMENTS] [ticalc.org] [Omnimaga] [Cemetech] [TI-calculator newsgroup]

You need a TI graphing calculator to run these programs. If you have a TI Connectivity Cable/TI-Graph Link you can download these programs into the calculator instead of typing them in. (I use a home built $4-link.) Documentation and ASCII listings are included. Now also with binaries for TI-82, TI-82 STATS, TI-83, TI-83 Plus & TI-84 Plus (including Silver Edition), see the links 82 and 82S/83/83+/84+, respectively. TI-82 STATS is more similar to TI-83 than TI-82 and use the 82S-links. TI-82 Advanced EDITION PYTHON is more similar to TI-83 Plus and use the 83+-links.

  • ARITM

    Aritm trains you in simple mental calculation. This program teaches its users the addition, subtraction, multiplication, and division tables. Aritm is also available for some CASIO-calculators, mobile phones w/ Android or Java ME, web browsers w/ Java SE, and OpenOffice/LibreOffice Calc, see http://aritm.orbin.se/.

    82 82S/83 83+/84+
  • STATS

    15 statistics programs for TI-82. Some are pedagogical, e.g. demonstrate the Central Limit Theorem. Others replace statistical tables, or do statistical graphing.

    82
  • DBD

    A days between dates game.

  • FACTOR

    This program factors integers into prime components.

    82 82S/83/83+/84+
  • FFT

  • NELDER

    Nonlinear minimization using the Nelder-Mead method. Useful for modeling and regression.

  • NORMAL

  • SPLINE v1.1

    You enter points graphically and then the program draws a smooth curve through them, a uniform cubic spline.

  • ROMAN v1.0

    Converts numbers in Arabic notation to Roman numerals. Does also find the minimal Roman form.

    82 82S/83/83+/84+
  • COMPLEX v1.0

    v1.0 of a practical complex number calculator. Menu-driven.

    82 82S/83/83+/84+
  • GCDLCM v1.0

    Computes the Greatest Common Divisor [or Highest Common Factor] of a LIST of integers. Also includes an LCM/LCD program.

    82 82S/83/83+/84+
  • BASECONV v1.0

    Converts numbers between different bases (radices).

    82 82S/83/83+/84+
  • QUEUESIM v1.0

    Simulates a bank queue graphically. Uses the negative exponential distribution.

    82 82S/83/83+/84+
  • TOCCATA v1.0

    A Mandelbrot set plotter. Runs in about 1 hour on the TI-82. ZOOMing is possible. Demonstrates 'End' problems.
    Toccata by J. S. Bach.
